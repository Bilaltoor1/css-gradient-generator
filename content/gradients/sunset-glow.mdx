# Sunset Glow Gradient

Experience the warmth and beauty of a stunning sunset with this vibrant gradient. Perfect for creating eye-catching designs that evoke feelings of warmth, energy, and optimism.

## Color Palette

This gradient features three carefully selected colors:

- **Burnt Orange** (#FF6B35) - The primary sunset color
- **Golden Orange** (#F7931E) - The warm middle tone
- **Sunny Yellow** (#FFD23F) - The bright highlight

## Usage Examples

### CSS Implementation

```css
background: linear-gradient(135deg, #FF6B35 0%, #F7931E 50%, #FFD23F 100%);
```

### Design Applications

- Website headers and hero sections
- Button backgrounds and call-to-action elements
- Social media graphics and banners
- Mobile app interfaces
- Brand identity elements

The warm tones in this gradient evoke:

- Pair with neutral colors for balance
