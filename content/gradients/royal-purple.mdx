# Royal Purple Gradient

Experience the elegance and sophistication of royalty with this stunning purple gradient. Ideal for luxury brands and premium designs.

## Color Palette

- **Indigo** (#4B0082) - Deep royal base
- **Blue Violet** (#8A2BE2) - Rich middle tone
- **Plum** (#DDA0DD) - Soft highlight

## Usage Examples

### CSS Implementation

```css
background: linear-gradient(135deg, #4B0082 0%, #8A2BE2 50%, #DDA0DD 100%);
```
